# coral-reef-detection

### setup

Download all data from [the drive](https://drive.google.com/drive/folders/1mOuhlo0y-b65eo8QzlyUYLQMpwmvJYXF) and put it in a data folder according to this structure:
```
data
├── coral_bleaching
│   ├── others
│   ├── reef_support
├── benthic_datasets
│   ├── point_labels
│   ├── mask_labels 
```
